# Event Management System

A simple web-based Event Management System that allows users to view events and register for them.

## Features
- Event listing
- User registration
- Data storage using browser localStorage
- Clean and simple UI

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository
2. Open `index.html` in your browser
3. Register for any event
