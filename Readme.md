# Event Management System

A professional web-based Event Management System that allows users to register for events and view registered participants.

## Features
- Event listing
- User registration form
- Firebase backend integration
- Admin-style registration dashboard
- Clean and responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
- Firebase Realtime Database

## How to Run
1. Clone the repository
2. Configure Firebase credentials in `script.js`
3. Open `index.html` or deploy via GitHub Pages
